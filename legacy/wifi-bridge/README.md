# The WiFi Bridge

> **Legacy product.** This is an earlier MeatStick generation. It is still supported in the MeatStick App, but it is no longer part of the current lineup. Looking for a current model? See [Start Here](../../README.md).

The WiFi Bridge is a companion device to the MeatStick that allows you to view your temps without your phone or allowing you to view your temps from far away.&#x20;
