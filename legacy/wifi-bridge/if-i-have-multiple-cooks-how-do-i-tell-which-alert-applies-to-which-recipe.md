# If I have multiple cooks, how do I tell which alert applies to which recipe?

#### Bridge LCD Alerts

* **Automatic Display:** When an alert is triggered, the Bridge LCD will automatically switch to show the recipe that needs attention.

#### Alert Prioritization

If multiple alerts are activated simultaneously, the system prioritizes them as follows:

1. **Target Temperature Reached**
2. **Early Warning Temperature Reached**
3. **Ambient Temperature Too High/Low**
4. **No Bluetooth Signal**
5. **Low Battery**
6. **New/Edited Recipe**

By following this intuitive setup, you'll be able to effortlessly keep track of alerts and ensure a smooth cooking process across all your recipes.
