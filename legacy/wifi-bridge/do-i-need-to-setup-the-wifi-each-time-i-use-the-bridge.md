# Do I need to set up the WiFi each time I use the Bridge?

* Once set up, the WiFi Bridge retains the information of the last connected WiFi network.
* No need for repeated configuration each time you use it.

#### Reconfiguration Process

* If the Bridge can't connect to the WiFi network during startup, it will broadcast its own WiFi signal.
* This allows you to connect it to a different WiFi network if needed.

This streamlined process ensures that your WiFi Bridge remains ready for seamless connectivity, enhancing your cooking experience.
