# Do I need to set up the WiFi each time I use the Smart Base?

Once you've set up the WiFi of your Smart Base, there's no need to repeatedly configure it every time you use it. The Base conveniently retains the information of the last connected WiFi network.

If the Base can't connect to a WiFi network during startup, the app will prompt you to connect it to a different network. When you set up the Base on a new network, the old WiFi settings will be replaced.

This streamlined process ensures that your Smart Base remains ready for seamless connectivity, enhancing your cooking experience.
