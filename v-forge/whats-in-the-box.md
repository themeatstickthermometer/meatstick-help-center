# What's in the V Forge box?

Every V Forge Set (and every multi-Stick variant) includes the same three core components:

| Item | What it does |
|---|---|
| **MeatStick V Core Stick** | The wireless Stick that goes in the meat. 5 internal temperature sensors, 50+ hour battery life, 1200°F ceramic handle. |
| **MeatStick V WiFi Hub** | The base station. Charges your Stick, extends Bluetooth range, and bridges to your WiFi network for cloud monitoring. Has a small port on one end for the Wired Ambient Probe. |
| **MeatStick V Wired Ambient Probe** | A thin stainless probe on a 1 m cable that clips at the grate to measure pit/oven temperature independently of the meat. Plugs into the WiFi Hub. |

Plus a **USB-C charging cable** for the WiFi Hub and a **quick-start guide**.

---

## Multi-Stick Forge variants

If you bought a V Forge Set with 2, 3, or 4 Sticks, you also get one or more **V Core Sets** in the box — each one is a separate V Core Stick paired with a stackable charger:

| Set | What's added on top of the standard Forge Set |
|---|---|
| V Forge Set — **2 Sticks** | + 1 V Core Set (Stick + stackable charger) + 3 colored silicone ID rings |
| V Forge Set — **3 Sticks** | + 2 V Core Sets + 6 colored silicone ID rings |
| V Forge Set — **4 Sticks** | + 3 V Core Sets + 9 colored silicone ID rings |

Each additional V Core Set is a complete, self-contained Stick — same Hub manages them all in the MeatStick App.

### About the silicone ID rings

Once you're running more than one Stick on the same cook, the colored slip-on rings (yellow / blue / red) help you tell them apart on the grill. You assign each Stick its color in the MeatStick App during first-time pairing — the in-app reading then matches the ring on the Stick.

## What if something's missing?

If your box is missing any of the components above, [fill out a ticket in the app](https://tms.fyi/help) and our Support Team will get you sorted.
