# Adding more Sticks to your V Forge

When one Stick isn't enough — multi-protein cooks, holiday spreads, competition rigs — the V Forge family scales by adding **V Core Sets**. Same WiFi Hub, same Wired Ambient Probe, same MeatStick App. You never re-buy the smart hardware.

## How it scales

| Set | Sticks total | What's added on top of the base V Forge Set |
|---|---:|---|
| V Forge Set | 1 | (the base set) |
| V Forge Set — **2 Sticks** | 2 | + 1 V Core Set |
| V Forge Set — **3 Sticks** | 3 | + 2 V Core Sets |
| V Forge Set — **4 Sticks** | 4 | + 3 V Core Sets |

Each additional V Core Set is a complete, self-contained Stick paired with a stackable charger — not just a bare Stick. That means each extra Stick can also be used as a stand-alone V Core thermometer on its own, paired directly to a phone over Bluetooth.

## Adding an extra Stick to an existing V Forge

If you already own a V Forge Set and want to add more Sticks later:

1. Buy a [V Core Set](../README.md) — it ships with its own Stick + stackable charger
2. Open the MeatStick App on the same account
3. Pair the new Stick (the app walks you through it)
4. Optionally assign it a color ring (yellow, blue, or red — three rings ship with every V Core Set)

The new Stick will appear alongside your existing one in the app. The same WiFi Hub manages both — no extra Hub to buy, no extra ambient probe.

## How the chargers stack

Each V Core Set ships with a **stackable charger** — a slim, USB-C-powered cradle. Multiple chargers lock together end-to-end in a row, and one USB-C cable at the end of the chain powers every charger. So if you have a V Forge Set with 3 Sticks, the two extra Core Set chargers click together side-by-side on the counter and share a single cable.

The WiFi Hub doesn't physically connect to the stackable chargers — it's a separate piece. The Hub stays on the side table; the stackable chargers stay on the counter for charging between cooks.

## Color ID rings — telling Sticks apart on the grill

Every V Core Set ships with **three colored silicone rings** (yellow, blue, red). When you're running more than one Stick on the same cook, slip a ring on each Stick's handle and assign it the matching color in the MeatStick App during pairing. The reading you see in the app will match the ring on the Stick — no more guessing which one is in the brisket and which is in the ribs.

| Bundle | Total color rings included |
|---|---:|
| V Forge Set — **2 Sticks** | 3 (one V Core Set) |
| V Forge Set — **3 Sticks** | 6 (two V Core Sets) |
| V Forge Set — **4 Sticks** | 9 (three V Core Sets) |

## Maximum Sticks per Hub

The WiFi Hub supports the full Forge bundle counts — up to 4 Sticks visible simultaneously in the app. Beyond that, you'd be running multiple Hubs, which is supported but uncommon for home cooks.
